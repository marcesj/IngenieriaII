Ejemplos de etiquetas block

1) Párrafos y divisiones

Las etiquetas block ocupan todo el ancho disponible y suelen organizar el contenido por bloques.

Ejemplos comunes: encabezados, párrafos, listas, tablas, divisiones y secciones.

Este contenido está dentro de un <div>, que también es una etiqueta block.
