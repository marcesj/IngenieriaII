Tecnicatura Superior en Desarrollo de Software

Repositorio correspondiente a la materia Ingeniería II de 2° año, donde se desarrollarán proyectos y prácticas utilizando tecnologías web fundamentales.

📚 Descripción

Este repositorio contiene los trabajos prácticos, ejercicios y proyectos realizados durante la cursada de la materia Ingeniería II.
El enfoque principal está en el desarrollo de aplicaciones web utilizando tecnologías del lado del cliente.

🚀 Tecnologías utilizadas

A lo largo de la materia trabajaremos con:

HTML5 → Estructura de páginas web
CSS3 → Estilos y diseño visual
JavaScript → Interactividad y lógica del lado del cliente
Bootstrap → Framework para diseño responsive y componentes UI

🎯 Objetivos de la materia
Comprender los principios del desarrollo web moderno
Aplicar buenas prácticas de diseño y programación
Desarrollar interfaces web responsivas
Integrar HTML, CSS y JavaScript en proyectos reales
Utilizar frameworks como Bootstrap para agilizar el desarrollo
